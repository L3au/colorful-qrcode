# Web Store Listings

## Chrome Web Store

**Title:** Colorful QRCode

**Summary:** Generate colorful QR codes for any page — works offline, no tracking.

**Description:**

Colorful QRCode generates a unique, colorful QR code every time you click the icon. Each code uses a randomly chosen dark color, so no two look the same.

Features:
• Instant QR code for the current page URL
• Random dark color on every open — always unique
• Site favicon displayed as a center logo on the QR code
• Click the QR code to edit text and generate a custom code
• Localhost URLs are automatically replaced with your LAN IP — perfect for testing on mobile devices
• Works completely offline — no data leaves your browser
• Lightweight and fast — no permissions beyond reading the active tab URL

Simply click the extension icon to get a scannable QR code. Click the code to switch to edit mode, type any text, and press Enter to generate.

**Category:** Developer Tools
